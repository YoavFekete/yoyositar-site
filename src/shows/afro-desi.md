# Afro-Desi

*A high-energy live set blending sitar, African string instruments, and rhythm — built for festivals and dance-driven stages.*

A dynamic meeting of **African strings (kora / goni traditions)** and **Indian raga**, carried by live improvisation, shared rhythm, and deep groove — hypnotic, emotional, and relentlessly alive.
